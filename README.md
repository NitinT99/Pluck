# Pluck

YouTube Audio & Video Downloader. Download MP3 audio or MP4 video from YouTube and YouTube Music.

## Features
- Download audio (MP3) up to 320kbps
- Download video (MP4) up to 2K
- Auto-fetches video thumbnail and metadata
- Works with YouTube and YouTube Music URLs

## Local Setup
1. Clone the repo
2. Create a virtual environment: `py -3.11 -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python app.py`
6. Open `http://localhost:5001`
